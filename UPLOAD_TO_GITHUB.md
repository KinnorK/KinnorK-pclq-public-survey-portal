# Uploading This Package to GitHub

1. Sign in to GitHub and create a new **private** repository named `pclq-survey-portal`.
2. Extract the ZIP on your computer.
3. Open the extracted `PCLQ_PUBLIC_CLOUD_FINAL` folder.
4. In the empty GitHub repository, choose **uploading an existing file**.
5. Drag all files and folders from inside `PCLQ_PUBLIC_CLOUD_FINAL` into the upload area.
6. Confirm that these files are at the repository root:
   - `render.yaml`
   - `Dockerfile`
   - `requirements.txt`
   - `app/`
7. Commit the upload.
8. Continue with `DEPLOY_RENDER.md`.

Do not upload a `.env` file containing passwords or secrets. The included `.env...example` files contain placeholders only.
