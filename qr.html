How it works

  1. A participant scans the QR code or opens the link.
  2. The participant completes the public questionnaire without logging in.
  3. The server generates a state-wise PCLQ ID and stores the response.
  4. Scores appear only in this restricted portal.
The QR code uses the active public HTTPS address. After adding a custom domain, set PUBLIC_BASE_URL to that domain before printing permanent materials.
