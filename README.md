# PCLQ Public Cloud Survey and Research Portal

Start with **README_FIRST_CLOUD.md**.

This repository contains a public no-login survey at `/survey`, a private Founder/Analyzer portal at `/login`, automatic PCLQ scoring, state-specific participant IDs, PostgreSQL storage, filters, reports, and exports.

Primary deployment configuration: **Render Blueprint (`render.yaml`)**.
Alternative deployment configuration: **Railway (`railway.toml`)**.

Local Windows files are in `local_testing/` and are only for development or same-Wi-Fi testing. They do not create a permanent internet address.
